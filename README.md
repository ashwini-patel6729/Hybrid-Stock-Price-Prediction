**Project Overview** 

This project presents an industry-oriented hybrid stock price prediction system that combines Deep Learning, Financial Sentiment Analysis, and Technical Indicators for forecasting future stock prices and market trends.

The system uses:

LSTM (Long Short-Term Memory) for time-series forecasting
FinBERT for financial news sentiment analysis
RSI, MACD, and VWAP technical indicators
Yahoo Finance API for historical stock market data
Streamlit frontend dashboard for real-time visualization

The objective of the project is to improve stock price prediction accuracy by integrating historical market data with financial sentiment analysis.# Hybrid-Stock-Price-Prediction.

**Machine Learning Model:-**

LSTM (Long Short-Term Memory)

LSTM is a deep learning algorithm designed for sequential and time-series forecasting tasks.

Why LSTM?
Learns long-term dependencies
Handles sequential stock market data efficiently
Performs well on time-series prediction
Reduces vanishing gradient problem



**FinBERT Sentiment Analysis:-**


FinBERT is a transformer-based Natural Language Processing (NLP) model trained specifically on financial text.

Purpose:
Analyze financial news headlines
Detect market sentiment
Improve stock trend prediction

Positive
